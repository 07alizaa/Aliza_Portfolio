## 🚀 Getting Started

### Prerequisites
- Node.js (v18 or higher)
- npm or yarn

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/07alizaa/Aliza_Portfolio.git
   cd Aliza_Portfolio
   ```

2. **Install frontend dependencies**
   ```bash
   cd frontend
   npm install
   ```

3. **Start the development server**
   ```bash
   npm run dev
   ```
   The application will be available at `http://localhost:5173`

## 🌐 Deployment

### Hosted on Vercel
The portfolio is automatically deployed on [Vercel](https://vercel.com) with the following setup:

- **Auto-deployment:** Deploys automatically on every push to the `main` branch
- **Environment:** Production-ready with global CDN
- **SSL:** Free SSL certificate included
- **Builds:** Optimized production builds with Vite

### Deploy Your Own

1. **Fork this repository**
2. **Connect to Vercel:**
   - Go to [vercel.com](https://vercel.com)
   - Click "New Project"
   - Select your GitHub repository
   - Click "Deploy"
3. **Done!** Your portfolio will auto-deploy on every push 🚀
